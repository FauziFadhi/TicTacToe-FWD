# tic-tac-toe

## Vue js installation
run this script if you dont have installed vuejs in you machine. if you already have it, just skip to the `Project Setup`
```
npm install -g @vue/cli
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
